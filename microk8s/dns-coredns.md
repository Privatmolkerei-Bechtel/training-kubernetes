# coredns/dns (microk8s) 

## Aktivieren

```
microk8s dns 

# Achtung dns wird erst bei neuen pods richtig eingetragen) 
```

## Deaktiveren 

```
microk8s disable 
```

## Aktiveren mit speziellen DNS - Servern 

```
micro8ks enable dns:1.1.1.1,2.2.2.2
# Ref: https://microk8s.io/docs/addon-dns
```
